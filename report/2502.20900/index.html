1. 论文速览

1.1 提出分层 VLA 框架：上层 VLM 负责语言目标分解和目标定位，下层 di

提出分层 VLA 框架：上层 VLM 负责语言目标分解和目标定位，下层 diffusion controller 负责灵巧手闭环抓取。

1.2 DexGraspVLA 将开放词汇灵巧抓取拆成两个频率不同的层级：低频 VL

DexGraspVLA 将开放词汇灵巧抓取拆成两个频率不同的层级：低频 VLM 负责理解自由语言指令、选择目标并检查任务是否完成；高频 diffusion controller 负责真正的多指接近、闭合与抬起。它的关键设计不是让大模型直接输出关节动作，而是用 SAM mask 与冻结的 DINOv2 特征把复杂视觉场景变成低层控制器更容易泛化的条件。对于首次复现灵巧手 VLA，它的模块边界清晰、问题定义集中，是较好的起点。

2. 研究问题与动机

2.1 现有灵巧抓取方法往往在单物体、固定背景或有限类别中训练，换到杂乱桌面、未见物

现有灵巧抓取方法往往在单物体、固定背景或有限类别中训练，换到杂乱桌面、未见物体和自由语言指令后容易失效。

2.2 直接微调通用 VLA 又需要大量覆盖视觉变化的机器人数据。

直接微调通用 VLA 又需要大量覆盖视觉变化的机器人数据。

2.3 DexGraspVLA 的动机是保留预训练视觉语言模型的语义与视觉泛化能力，

DexGraspVLA 的动机是保留预训练视觉语言模型的语义与视觉泛化能力，只让低层策略学习接触丰富的抓取动作，从而用有限真实示范覆盖更大的物体、背景和光照分布。

2.4 系统采用高低层分工：低频 VLM planner 解析自由语言、提出子任务、

系统采用高低层分工：低频 VLM planner 解析自由语言、提出子任务、定位目标并检查任务完成；高频 diffusion controller 负责连续多指动作。这样的分层避免让大模型直接承担精细闭环控制。

长程任务由高层逐物体调用低层控制器完成。

6. 复现要点

6.1 优先复现单物体抓取，再接入高层 planner。

优先复现单物体抓取，再接入高层 planner。

需要多指手与机械臂、第三视角和腕部相机、20Hz 左右控制链路、遥操作示范、SAM/DINOv2、可进行视觉 grounding 的 VLM，以及 diffusion action-chunk 训练代码。

最容易出问题的是 mask 跟踪、相机与 proprioception 时间对齐、不同硬件动作维度，以及 VLM prompt 与原论文不一致导致的目标框错误。

先复现单目标 mask-conditioned diffusion controller，再接入高层 VLM planner；低层链路跑通之前不应直接挑战完整长程任务。

7. 分析、局限与边界

7.1 模块化结构使错误来源可定位，但也产生明显级联风险：VLM 选错物体、SAM

模块化结构使错误来源可定位，但也产生明显级联风险：VLM 选错物体、SAM 分错 mask 或 tracker 丢失目标，低层控制再强也无法恢复。

7.2 实验主要围绕桌面抓取、清理和先推后抓，尚未证明工具使用、双手协作、透明/强反

实验主要围绕桌面抓取、清理和先推后抓，尚未证明工具使用、双手协作、透明/强反光物体或精细力控能力。

7.3 90% 以上成功率依赖特定硬件、重试策略和测试协议，不能直接外推到任意灵巧手

90% 以上成功率依赖特定硬件、重试策略和测试协议，不能直接外推到任意灵巧手。

7.4 我的阅读判断：适合作为灵巧手 VLA 复现起点。建议先跑通 mask-con

我的阅读判断：适合作为灵巧手 VLA 复现起点。建议先跑通 mask-conditioned 低层 controller，再替换 VLM 或视觉特征做创新；不要一开始就复现完整长程系统。与 DexVLA 比，它更专注灵巧抓取；与 UniHM 比，它的高低层职责更清楚。
